#Bricks
Bricks is a tetris game developed using javascript, css and html5.


#Example and demo
Place the files in a folder and run the example.html, with an modern browser.

You can also see a demo of the game on [http://branduren.com/projects/bricks/](http://branduren.com/projects/bricks/)
